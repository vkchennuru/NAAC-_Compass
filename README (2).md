## 📋 What Is NAAC Compass?

NAAC Compass is a **single-file web application** that guides any Indian college — regardless of technical knowledge — through the complete **NAAC Binary Accreditation framework** (10-attribute system, 2024 onwards).

Designed for:
- **IQAC Coordinators** — manage all 10 attributes, track compliance, assign tasks
- **Faculty Members** — know exactly what activities to perform and what documents to prepare
- **Principals & Management** — view compliance status and generate SSR summaries
- **Admin / Non-Teaching Staff** — simple checklists for infrastructure, finance, and green initiatives

## 🚀 Quick Deploy (5 Minutes)

### Step 1 — Get the file
Download `index.html` from this repository.

### Step 2 — Create GitHub account
Go to [github.com/signup](https://github.com/signup) — it is free.

### Step 3 — Create repository
- Click **+ New repository**
- Name: `naac-compass`
- Set to **Public**
- Tick ✅ **Add a README file**
- Click **Create repository**

### Step 4 — Upload index.html
- In your repository → **Add file → Upload files**
- Drag `index.html` → **Commit changes**

### Step 5 — Enable GitHub Pages
- **Settings → Pages → Branch: main → / (root) → Save**
- Wait 2 minutes → your site is live at:

```
https://YOUR-USERNAME.github.io/naac-compass/
```

## 🔒 Privacy

All college data is stored **only in your browser** (localStorage). No data is sent to any server except when using the AI chat feature (which uses the Anthropic API).

## 📖 How to Use

1. Open the tool URL in any browser
2. Select your role (IQAC Coordinator / Faculty / Principal / Admin)
3. Click **"Start Setup Wizard"** for a guided 7-step onboarding
4. Use the **Dashboard** to explore all 10 attributes
5. Click any attribute card → see What To Do, What To Document, How To Report
6. Use **📋 Tracker** to assign tasks to faculty and track completion
7. Use **📝 AQAR** to auto-draft your Annual Quality Assurance Report
8. Use **📁 DVV** to prepare your evidence files attribute-wise
9. Use **📋 SSR** to build your Self Study Report summary
10. Use **🤖 NAAC Buddy** (floating button) to ask any question
