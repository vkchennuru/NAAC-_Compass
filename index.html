🚀 Deployment Steps
1
Download Your NAAC Compass File
You already have the complete single-file app. Click the button below to download it, then rename it to index.html.

2
Create a Free GitHub Account
Go to github.com → Click "Sign up" → Create your account (free). Choose a username like dr-krishnaveni or your college name.

→ Open GitHub Sign Up ↗
3
Create a New Repository
After logging into GitHub:
  • Click the + New button (top right)
  • Repository name: naac-compass
  • Set to Public
  • Tick ✅ Add a README file
  • Click Create repository
4
Upload Your index.html File
Inside your new repository:
  • Click Add file → Upload files
  • Drag your index.html file into the box
  • Scroll down → click Commit changes
That's it — your file is now on GitHub!
5
Enable GitHub Pages (Free Hosting)
In your repository:
  • Click Settings (top menu)
  • Left sidebar → Pages
  • Under "Branch" → select main → folder / (root)
  • Click Save
  • Wait 1–2 minutes → your site is live! ✅
Your URL: https://YOUR-USERNAME.github.io/naac-compass/
6
Share with Other Colleges
Once live, share the URL via:
  • 📱 WhatsApp — IQAC Coordinators group
  • 📧 Email — to Principals and HODs
  • 🌐 College website — add as a resource link
  • 📝 Blog — post on cvkrishnaveni.blogspot.com
  • 🎓 Education communities — share with IQAC networks and NAAC forums

💡 Pro Tip: When you update the tool (new phases, content changes), just upload the new index.html to the same GitHub repo — the live site auto-updates within minutes. No server restarts, no maintenance!
7
Add Gemini API Key (for AI Features)
The AI Buddy and AQAR auto-draft currently use the Claude API. For a public deployment, you may want to use the free Gemini API instead (no usage limits for basic use):
  • Go to aistudio.google.com/apikey ↗
  • Click Create API Key (free, no credit card)
  • Replace API calls in the code with Gemini endpoint
  • Or keep the current Claude API — works great for institutional use
